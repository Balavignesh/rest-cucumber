# rest-cucumber
 
This is the basic app which integrates the following technologies


1. Java 8
2. Spring Boot
3. H2 Database
4. Spring Jpa
5. Spring MockMvc
6. Spring Scheduler
7. Lombok
8. Actuator
9. Junit 5
10. Cucumber
11. Masterthought - cucumber-reporting
12. Jacoco
13. TestRestTemplate
