# rest-cucumber
 
This is the basic app which integrates the following technologies


1. Spring Boot
2. H2 Database
3. Spring Jpa
4. Spring MockMvc
5. Lombok
6. Actuator
7. Junit 5
8. Cucumber
9. Masterthought - cucumber-reporting
10. Jacoco
11. TestRestTemplate
