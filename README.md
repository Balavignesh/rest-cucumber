# rest-cucumber
 
This is the basic app which integrates the following technologies


1. Spring Boot
2. H2 Database
3. Spring Jpa
4. Spring MockMvc
5. Spring Scheduler
6. Lombok
7. Actuator
8. Junit 5
9. Cucumber
10. Masterthought - cucumber-reporting
11. Jacoco
12. TestRestTemplate
